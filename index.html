Mixins for simple (bordered!) popups.

  • Top, Bottom, Left, and Right.
  • Include the popup.scss file and include the mixin with optional parameters in your SCSS/SASS file like so: @include popout-bottom('FILL COLOR', 'BORDER COLOR', 'MIN-WIDTH','MAX-WIDTH');
  • Use javascript or media queries to move them around more dynamically.
  • Or simple simple CSS :hover/:active psuedo classes to show them.
  • The world is your oyster.
